# Optimización de Data Warehouse

Este repositorio contiene la evidencia, estructura y documentación correspondiente a la tarea **“Optimización de Data Warehouse”**, desarrollada para la materia **Almacenamiento de Grandes Bases de Datos**, dentro de la **Maestría en Ciencia de los Datos** de la Universidad de Guadalajara.

## Objetivo

Diseñar y documentar una estrategia de optimización para un Data Warehouse, considerando principios de almacenamiento eficiente, particionamiento, organización de datos y consultas analíticas sobre grandes volúmenes de información.

## Contexto

El trabajo se enfoca en el análisis y optimización de estructuras de almacenamiento para grandes bases de datos, tomando como referencia escenarios donde se requiere consultar información de forma eficiente mediante particiones, organización lógica de archivos y buenas prácticas de diseño en entornos de procesamiento distribuido.
